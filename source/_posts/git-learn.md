# git 基本操作
## git 安装

```
yum -y install git 
apt-get install git-all
```

## 基本命令
1. 查询版本
```
git --version
```

2. 初始化本地仓库
```
git init
```
3. 查询状态
```
git status
```
4. 添加到占存区
```
git add <file>
git add -A // 全部提交
```
5. 移除本地仓库 ues .. to unstage
```
git rm --cached <file> 
```
6. 提交给本地服务器
```
git commit -m "日志"
```
7. 提交服务器
